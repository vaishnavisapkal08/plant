💡 How this database works:

This table is dynamically fetching live data from the GET /plants API endpoint via your Node.js (Express) server.

Where is the database located?
It is currently stored as a JSON file named plants.json in the root folder of this codebase (c:\pr1\plants.json).
How do I add a new plant?
Simply open `plants.json` in any text editor or IDE and add a new block to the array, following the same format. Once you save it, this dashboard will immediately reflect the new data!
